## 研究边界

RAGFlow 研究重点是数据进入系统后的工程质量：文档解析、chunking、hybrid retrieval、rerank、引用溯源、评测、治理和 Agent 上层边界。不要把产品宣传词直接当结论，必须降级成可验证假设。
