# 稳定知识

`knowledge/` 是稳定知识层，放跨来源整理后的 evergreen 文档。

这里的文件应该能被未来的自己和 LLM 直接复用。单篇视频总结、文章摘要或课程笔记不要放这里，除非已经被重写成稳定主题知识。

## 当前目录

- [ai/](ai/)：LLM、Agent、RAG、MCP、模型生态和 AI 工程知识。
- [meta/](meta/)：知识库结构、Karpathy 式知识库原则和仓库方法论。

## 进入标准

内容进入 `knowledge/` 前至少满足：

- 跨来源，或来自一个足够权威且已经复核的来源；
- 不是单纯摘要，而是被重写成可复用知识；
- 有来源入口；
- 明确适用边界；
- 标注未验证事项；
- 能被后续输出直接调用。

新增稳定知识优先使用 [knowledge 模板](../methods/templates/knowledge.md)。
