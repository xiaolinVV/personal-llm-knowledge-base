## 使用原则

RAGFlow 在本仓库里不是“万能知识库产品”的代名词。它的研究价值主要在文档解析、chunking、hybrid retrieval、rerank、引用溯源、评测和治理。只有当要验证具体工程判断时，才升级到 `labs/`。
