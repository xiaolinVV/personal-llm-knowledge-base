# Obsidian 使用说明

这个目录保存 Obsidian 视图层：MOC、Base 和使用规则。它不改变 `raw/`、`wiki/`、`schema/` 的真实归档规则。

## 使用原则

- 真实文件归属仍先按三层结构判断。
- 中文可读性放在首页、MOC、主题地图和链接文本里。
- 不为了 Graph 漂亮而机械加双链。
- 新文件优先使用 `schema/templates/` 下的模板。
- `.obsidian/` 默认不进 Git，设备布局和主题偏好不作为知识库正文。
