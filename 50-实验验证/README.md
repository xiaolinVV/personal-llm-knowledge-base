# 实验验证

`50-实验验证/` 是验证层，只放能验证具体判断的最小实验。

实验不是仓库主角。没有明确验证问题的代码，不进 `50-实验验证/`。

## 与其他目录的边界

- 资料来源放 `../10-来源索引/`。
- 学习笔记放 `../20-资料笔记/`。
- 稳定知识放 `../30-稳定知识/`。
- 结构化研究放 `../40-问题研究/`。
- 实验结论如果稳定，要回写到 `../30-稳定知识/` 或 `../40-问题研究/`。

## 当前目录

- [openai/](openai/)：OpenAI Responses API、Agents SDK、工具调用、RAG、handoffs、guardrails、evals、sandbox 等最小实验。
- [anthropic/](anthropic/)：Anthropic / Claude 相关实验入口。

## 每个实验必须包含

- `README.md`：目标、依赖、运行方式、验收标准。
- 最小可运行代码。
- 明确的验证命令。
- 实验结论或复盘链接。
- 未验证事项。

新增实验优先使用 [lab 模板](../60-方法库/templates/lab.md)。
